# To-Do-List 

📝 To-Do List App
A fully functional To-Do List web application built using HTML, CSS, JavaScript, and Vite as the build tool.

🔧 Features
✅ Add Tasks

❌ Delete Tasks

✏️ Edit Tasks

✔️ Mark Tasks as Completed (Toggle)

🕒 Timestamp for when tasks are added

🚀 Technologies Used
# HTML5
# CSS3
#  Vanilla JavaScript
# Vite
